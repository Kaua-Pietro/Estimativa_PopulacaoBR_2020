# 🇧🇷 População Brasileira 2020

Projeto de visualização de dados desenvolvido com HTML, CSS, JavaScript e Vega-Lite, utilizando dados de população estimada dos municípios brasileiros em 2020.

## 📊 Funcionalidades

- População total por estado
- Top 10 municípios mais populosos
- Distribuição populacional dos municípios
- Rankings e indicadores demográficos
- Visualizações interativas

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
- Vega-Lite
- Python + Pandas

## 📁 Fonte dos Dados

Dados públicos do IBGE referentes à estimativa populacional dos municípios brasileiros em 2020.
